# Images Needed for Random Item Page

Please add the following 5 constellation images (at least 300x300 pixels, approximately square) to the `public/images` folder:

1. orion.jpg - Image of Orion constellation
2. ursamajor.jpg - Image of Ursa Major constellation  
3. cassiopeia.jpg - Image of Cassiopeia constellation
4. crux.jpg - Image of Crux (Southern Cross) constellation
5. andromeda.jpg - Image of Andromeda constellation

These images will be randomly selected and displayed on the /randomitem page.

You can find constellation images from:
- NASA image library
- Wikimedia Commons
- Stellarium screenshots
- Astronomy websites (with proper licensing)
